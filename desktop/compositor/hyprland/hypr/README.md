# hypr

To include each of the module files in this directory for ```hyprland.conf```, enter the following there: ```source = path/to/file```

Blank:
```hyprland.conf
source = ~/.config/hypr/general.conf
source = ~/.config/hypr/decoration.conf
source = ~/.config/hypr/animations.conf

source = ~/.config/hypr/keybindings.conf

source = ~/.config/hypr/layerrules.conf
source = ~/.config/hypr/windowrules.conf
source = ~/.config/hypr/misc.conf
```
